# 档案检索系统

## 项目简介
档案检索系统是一款专业的档案管理软件，专为政府机构、企事业单位设计，用于高效管理和检索各类电子档案。本系统提供完整的功能集和专业的技术支持。

## 主要功能
- **档案导入与管理**：支持多种格式档案的批量导入和管理
- **高效检索**：强大的全文检索和多条件筛选功能
- **数据安全**：自动备份和数据恢复机制
- **用户权限**：完善的用户权限管理系统
- **报表生成**：灵活的统计报表生成功能

## 系统要求
- Windows 7/10/11 64位操作系统
- 4GB 内存以上（推荐8GB）
- 1GB 可用磁盘空间
- 1366×768 以上屏幕分辨率
